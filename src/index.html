InterShare

A simple and easy to use cross-platform app to share files between devices. While it sends the data over a network connection, it does not require any internet connection. Data is transferred directly from one device to another.


Download Source Code

Important

This is a very early pre-release version. Things may change drastically and may not be backwards compatible. Check the website regularly for new updates.

Secure

Before the data is sent, InterShare establishes a secure connection between the devices. This ensures that the data is only accessible to the rightful recipient.

Portable

InterShare was designed from the beginning to be a tool that requires no installation or configuration. Just download the program and you're ready to go.

No Internet

No internet connection is needed. You can even use your mobile hotspot for transfer without spending any cellular data. Support for Bluetooth is also planned.

Cross-Platform

InterShare is available on every major platform. Linux, Windows, macOS, iOS and Android.
